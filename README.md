# big-data-ml
# 🏠 Предсказание медианной цены дома с помощью Linear Regression на Apache Spark и MLlib

**Пет-проект по машинному обучению на больших данных**

Проект демонстрирует построение модели линейной регрессии для предсказания медианной стоимости домов в жилом массиве (California Housing). Всё реализовано с использованием **PySpark** и библиотеки **MLlib** — от предобработки данных до обучения модели и оценки качества.

---

## ✨ Основные возможности

- Масштабируемая обработка данных с помощью Apache Spark
- Полный пайплайн машинного обучения на **MLlib** (VectorAssembler, StandardScaler, LinearRegression)
- Оценка модели: RMSE, MAE, R²

---

## 🛠 Технологии

- **Apache Spark** (PySpark)
- **MLlib** (модуль машинного обучения Spark)
- **Pandas** — для EDA
- **Jupyter Notebook**
